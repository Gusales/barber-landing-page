# Barber Landing Page

Página inicial de uma barbearia.

![App Screenshot](./.github/thumb.png)

## Stack utilizada

**Front-end:** HTML, SCSS.

**Outros:** Figma, Git e Github.

## Funcionalidades

- Página de contatos;
- Mapa de localização interativo;
- Link de conversa para WhatsApp;
- Animações e adaptado para diversos dispositivos.
